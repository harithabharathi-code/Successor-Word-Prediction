# Successor-Word-Prediction
This project predicts the next probable word using NLP techniques.  
The system uses Markov Models with Bigram and Trigram frequency analysis for prediction.  
An additional LSTM deep learning model is also implemented to improve contextual prediction accuracy.  

Technologies Used:
- Python
- NLP
- TensorFlow/Keras

Web application:
https://drvjds.shinyapps.io/PredictWord/
